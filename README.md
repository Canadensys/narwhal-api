narwhal-api
===========

API for the narwhal-processor

# Build Status

[![Build Status](https://travis-ci.org/Canadensys/narwhal-api.png)](https://travis-ci.org/Canadensys/narwhal-api)

# Run Locally
```
./gradlew -daemon clean runLocal
```

# Run Tests
```
./gradlew clean test
```

 * [GeoTools](http://www.geotools.org/) 10
